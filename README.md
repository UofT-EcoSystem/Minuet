# Minuet

Minuet is a library that efficiently implements sparse 
convolutions for point clouds on GPUs.

## Installation

```shell
pip3 install "torch~=2.1" "packaging~=23.2"
CMAKE_BUILD_PARALLEL_LEVEL=$(nproc) pip3 install .
```
